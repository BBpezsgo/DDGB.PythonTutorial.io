Készítők Források

Szintaxis

A behúzás a kódsor elején lévő szóközökre vonatkozik.
Ahol más programozási nyelvekben a kód behúzása csak az olvashatóságot szolgálja, a Pythonban a behúzás nagyon fontos.
A Python behúzást használ a kódblokk jelzésére.

if 5 > 2:
  print("5 nagyobb mint 2")

Ne így csináld!

if 5 > 2:
print("5 nagyobb mint 2")

A szóközök számát Te, mint programozó dönti el, de legalább egynek kell lennie.

if 5 > 2:
  print("5 nagyobb mint 2")
if 5 > 2:
    print("5 nagyobb mint 2")

Ne így csináld!

if 5 > 2:
  print("5 nagyobb mint 2")
    print("5 nagyobb mint 2")

Ugyanabban a kódblokkban ugyanannyi szóközt kell használnod, különben a Python hibát ad.

Megjegyzések

Megjegyzések használhatók a forráskód magyarázatára, ezzel olvashatóbbá tehetjük a kódot.
A megjegyzések segítségével megakadályozható a végrehajtás a kód tesztelésekor.

Megjegyzés létrehozása

A megjegyzések '#' karakterrel kezdődnek, így a Python figyelmen kívül hagyja őket:

#Ez egy megjegyzés
print("Hello, World!")

A megjegyzések elhelyezhetők egy sor végén, és a Python figyelmen kívül hagyja a sor többi részét:

print("Hello, World!") #Ez egy megjegyzés

A megjegyzéseknek nem kell szövegnek lennie, amely elmagyarázza a kódot, hanem arra is használható, hogy megakadályozza a Python kód végrehajtását:

print("Hello, World!") #Ez egy megjegyzés

Többsoros megjegyzés létrehozása

A Python nem igazán rendelkezik szintaxissal a többsoros megjegyzésekhez. Többsoros megjegyzés hozzáadásához #minden sorhoz beszúrhat egyet:

#Ez egy megjegyzés
#több sorba van
#írva mint egy.
print("Hello, World!")

Vagy, nem egészen a szándék szerint, használhat többsoros karakterláncot. Mivel a Python figyelmen kívül hagyja azokat a karakterlánc-literálokat, amelyek nincsenek hozzárendelve egy változóhoz, hozzáadhat egy többsoros karakterláncot (háromszoros idézőjelek) a kódhoz, és elhelyezheti benne megjegyzését:

"""Ez egy megjegyzés
több sorba van
írva mint egy."""
print("Hello, World!")

Mindaddig, amíg a karakterlánc nincs változóhoz rendelve, a Python beolvassa a kódot, de figyelmen kívül hagyja, és többsoros megjegyzést tett.

Változók

A változók adatértékek tárolására szolgáló tárolók.

Változók létrehozása

A Pythonnak nincs parancsa a változók deklarálására. A változó abban a pillanatban jön létre, amikor először értéket rendel hozzá.

x = 5
y = "Sanyi"
print(x)
print(y)

A változókat nem kell semmilyen konkrét típussal meghatározni, sőt a beállításuk után változtathatnak is.

x = 5 # 'x' most egy szám
x = "Sanyi" # 'x' most már egy szöveg
print(x)

Átkonvertálás

Ha egy változó adattípusát meg szeretnénk változtatni, ezt 'casting' segítségével megtehetjük.

x = str(3) # x most szöveg ('3')
y = int(3) # y most egész szám (3)
z = float(3) # z most törtszám (3.0)

Adattípus lekérése

Egy változó adattípusát a type() függvénnyel lehet kelérni.

x = 5
y = "Sanyi"
print(type(x))
print(type(y))

Egyszeres vagy dupla idézetek?

A karakterlánc-változók egyszeres vagy dupla idézőjelek használatával deklarálhatók:

x = "Sanyi"
# ugyan az mint
x = 'Sanyi'

Kis-nagybetű érzékenység

A változónevek megkülönböztetik a kis- és nagybetűket.
a = 4
A = "Sanyi"
# 'A' nem ugyan az a változó, mint az 'a'

Számok

A Pythonban három numerikus típus létezik:

  • int
  • float
  • complex
Numerikus típusú változók jönnek létre, amikor értéket rendel hozzájuk:

x = 1 # int
y = 2.8 # float
z = 1j # complex

Int (egész szám)

Az int vagy egész szám korlátlan hosszúságú egész szám, pozitív vagy negatív, tizedesjegyek nélkül.

x = 1
y = 35656222554887711
z = -3255522
print(type(x))
print(type(y))
print(type(z))

Float (törtszám)

A törtszám egy vagy több tizedesjegyet tartalmazó pozitív vagy negatív szám.

x = 1.10
y = 1.0
z = -35.59
print(type(x))
print(type(y))
print(type(z))

A float tudományos számok is lehetnek "e"-vel, amely a 10 hatványát jelzi.

x = 35e3
y = 12E4
z = -87.7e100
print(type(x))
print(type(y))
print(type(z))

Complex (összetett szám)

Az összetett számokat "j"-vel írjuk fel képzeletbeli részként:

x = 3+5j
y = 5j
z = -5j
print(type(x))
print(type(y))
print(type(z))

Típus Konverzió

Az egyik típusból a másikba konvertálhat a int(), float(), és complex() metódusokkal:

x = 1 # int
y = 2.8 # float
z = 1j # complex

#convert from int to float:
a = float(x)

#convert from float to int:
b = int(y)

#convert from int to complex:
c = complex(x)

print(a)
print(b)
print(c)

print(type(a))
print(type(b))
print(type(c))
A komplex számokat nem alakíthatja át más típusú számokká.

Véletlen szám

A Python nem rendelkezik random() véletlen számok létrehozására szolgáló funkcióval, de a Pythonnak van egy beépített modulja a "random", amely véletlen számok létrehozására használható:

import random

print(random.randrange(1, 10))

String-ek

A Pythonban szereplő szövegek vagy szimpla idézőjelekkel vagy kettős idézőjelekkel vannak körülvéve.
A 'hello' ugyanaz, mint a "hello".
Egy szöveg a következő 'print()' függvénnyel jeleníthet meg:

A szöveg hozzárendelése a változóhoz úgy történik, hogy a változó nevét egy egyenlőségjel követi és a szöveg:

Bemenet-kimenet

Kimenet

Ha ki szeretnénk írni egy szöveget vagy változót, azt a 'print' függvénnyel tehetjük meg.

Bemenet

Ha a felhasználótól be szeretnénk kérni egy adatot, azt az 'input' függvénnyel tehetjük meg.

A 'print' függvény csak szöveg változót tud adni. Ha számot szeretnénk bekérni, akkor a szöveget át kell konvertálni számmá:

Boolean-ok (logikai értékek)

A programozás során gyakran tudnod kell, hogy egy kifejezés 'True' (igaz) vagy 'False' (hamis). Bármilyen kifejezést kiértékelhet a Pythonban, és megkaphatja a két válasz egyikét, True vagy False. Két érték összehasonlításakor a rendszer kiértékeli a kifejezést, és a Python a logikai választ adja vissza:

print(10 > 9)
print(10 == 9)
print(10 < 9)

Értékek és változók értékelése

A 'bool()' függvény lehetővé teszi bármely érték kiértékelését, és cserébe, 'True' vagy 'False'

print(bool("Hello"))
print(bool(15))

A legtöbb érték igaz

Szinte minden értéket a rendszer arra értékel hogy van-e valamilyen tartalma.

  • Bármely szöveg True, kivéve az üres szövegeket.
  • Bármely szám az True, kivéve 0.
  • Bármely lista, sor, halmaz és szótár True, kivéve az üreseket.

# A következőek mind True-t adnak vissza
bool("abc")
bool(123)
bool(["alma", "körte", "banán"])

Egyes értékek hamisak

Valójában nem sok olyan érték van, amely kiértékelve False, kivéve az üres értékeket, mint például a (), [], {}, "", a 0 és az érték None. És persze az False érték kiértékítve False.

  • Bármely szöveg True, kivéve az üres szövegeket.
  • Bármely szám az True, kivéve 0.
  • Bármely lista, sor, halmaz és szótár True, kivéve az üreseket.

bool(False)
bool(None)
bool(0)
bool("")
bool(())
bool([])
bool({})

Elágazások

Az if utasítás mellett egy olyan kifejezés következik, aminek van logikai értéke. A number > 100 kifejezés, ha a number értéke nagyobb mint 100, akkor True, amúgy False kiértékelési értéket kap. A sort mindig a : zárja, erre figyeljünk.

number = int(input("Írj be egy számot: "))
if number > 100:
  print("A szám nagyobb mint 100")

Az else és az if egy párt alkotnak, ezért azonos bekezdést kell alkalmazni előttük (azonos bekezdési szintent (indentation level) kell lenniük).

number = int(input("Írj be egy számot: "))
if number > 100:
  print("A szám nagyobb mint 100")
else:
  print("A szám nem nagyobb mint 100")

Ha 99-et írunk be, akkor az első feltétel (number > 100) hamis lesz, így ugrunk a következőre (number == 100), de ez is hamis.Ha 100-at írunk be, akkor az első feltétel (number > 100) hamis lesz, így ugrunk a következőre (number == 100) ami igaz lesz,

number = int(input("Írj be egy számot: "))
if number > 100:
  print("A szám nagyobb mint 100")
elif number == 100:
  print("A szám pont 100")
else:
  print("A szám kissebb mint 100")

Listák

mylist = ["apple", "banana", "cherry"]

A listák több elem tárolására szolgálnak egyetlen változóban.
A listák a Pythonban az adatgyűjtemények tárolására használt 4 beépített adattípus egyike, a másik 3 pedig a Tuple , Set és Dictionary , amelyek mindegyike eltérő minőségű és felhasználású.
A listák szögletes zárójelekkel jönnek létre.

thislist = ["Alma", "Banán", "Cseresznye"]
print(thislist)

A listaelemek rendezettek, változtathatók, és engedélyezik az értékek megkettőzését.
A listaelemek sorszámozva vannak, az első elem sorszáma [0], a második elem sorszáma [1], stb.

Ha azt mondjuk, hogy a listák sorrendben vannak, az azt jelenti, hogy a tételeknek meghatározott sorrendje van, és ez a sorrend nem változik.
Ha új elemeket ad hozzá egy listához, az új elemek a lista végére kerülnek.

Vannak olyan listamódszerek, amelyek megváltoztatják a sorrendet, de általában: a tételek sorrendje nem változik.

Ugyan olyan elemek

Mivel a listák sorszámozottak, a listákban lehetnek azonos értékű elemek.

thislist = ["Alma", "Banán", "Cseresznye", "Alma", "Cseresznye"]
print(thislist)

A lista hossza

A lista hány elemének meghatározásához használja a következő len() függvényt.

thislist = ["Alma", "Banán", "Cseresznye", "Alma", "Cseresznye"]
print(len(thislist))

list1 = ["Alma", "Banán", "Cseresznye"]

A lista különböző típusú adatokat tartalmazhat.

list1 = ["abc", 34, True, 40, "Banán"]

type()

A Python szemszögéből a listák 'list' adattípusú objektumokként vannak definiálva.

mylist = ["Alma", "Banán", "Cseresznye"]

A list() konstruktor

Lehetőség van a list() konstruktor használatára is új lista létrehozásakor.

For

A for és in között szerepel annak a változónak (vagy változóknak!) a neve, amin keresztül el akarjuk érni az aktuális elemet (ez a példában a name nevet kapta). Itt nem szerepelhet másféle kifejezés. Az in után következik az objektum neve (vagy tetszőleges kifejezés), amit iterálunk (a példában ez a people változó)

people = ["Sanyi", "Bob", "Anita"]

for name in people:
  print(name)

While

A while melletti feltételétől függ, hogy meddig kell a while blokkját ismételtetni. Ezt magyarul bentmaradási feltételnek szokták hívni.

A sums változóban tároljuk majd a részösszeget, (tehát minden iterációban ehhez a változóhoz adjuk majd a begépelt számokat). Ez után kiírjuk az Írjon be egy számot: üzenetet, és a begépelt szöveget eltároljuk a next_number változóban, majd következik is a ciklus.

sums = 0

next_number = input("Írj be egy számot: ")
while next_number != "":
  sums = sums + float(next_number)
  next_number = input("Írj be egy számot: ")

print("Az összeg:", sums)

Ez egy igen béna kis példa, de a lényeg benne van: ha a counter változó értéke 2-vel osztható, akkor kiírjuk hogy páros és indítjuk is az új iterációt. Ha az érték páratlan, akkor sosem lépünk be a feltétel törzsébe, és megjelenik a páratlan felirat. A "break"paranccsal azonnal befejeződik a ciklus.

counter = 0

while counter <= 10:
  counter += 1

  if counter % 2 == 0:
    print(counter, "páros")
    continue
  print(counter, "páratlan") print("Az összeg:", sums)

Class-ok

A Python egy objektum orientált programozási nyelv. A Pythonban szinte minden objektum, tulajdonságaival és módszereivel együtt. Az osztály olyan, mint egy objektumkonstruktor, vagy egy "tervrajz" az objektumok létrehozásához.

Class létrehozása

class MyClass:
  x = 5

Objektum létrehozása

p1 = MyClass()
print(p1.x)

Az __init__() függvény

A fenti példák osztályok és objektumok a legegyszerűbb formájukban, és nem igazán hasznosak a valós alkalmazásokban.
Az osztályok jelentésének megértéséhez meg kell értenünk a beépített __init__() függvényt.
Minden osztálynak van egy __init__() nevű függvénye, amely mindig végrehajtásra kerül az osztály indításakor.
Használd az __init__() függvényt, hogy értékeket rendeld az objektum tulajdonságaihoz, vagy más műveleteket, amelyeket az objektum létrehozásakor kell elvégezni.

class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

p1 = Person("John", 36)

print(p1.name)
print(p1.age)

Try...Except

A try blokk lehetővé teszi egy kódblokk tesztelését hibák keresésére.
Az except blokk segítségével kezelheted a hibát.
A finally blokk lehetővé teszi a kód végrehajtását, függetlenül a try- és except blokkok eredményétől.

Kivételkezelés

Ha hiba történik, vagy kivétel(ahogy mi hívjuk), a Python általában leáll, és hibaüzenetet generál.
Ezek a kivételek a try utasítással kezelhetők:

try:
  print(x)
except:
  print("Kivétel történt")

Mivel a try blokk hibát jelez, a except blokk végrehajtásra kerül.
A try blokk nélkül a program összeomlik, és hibát jelez:

Ez az utasítás hibát jelez, mivel az x nincs definiálva:
print(x)

Több kivétel

Tetszőleges számú kivételblokkot definiálhatsz, pl. ha egy speciális kódblokkot szeretnél végrehajtani egy speciális hiba esetén:

# Írjon ki egy üzenetet, ha a try blokk NameError-t ad, és egy másikat egyéb hibák esetén:
try:
  print(x)
except NameError:
  print("Az x változó nincs definiálva")
except:
  print("Valami más hiba történt")

Else

Az else kulcsszó segítségével megadhat egy kódblokkot, amelyet akkor kell végrehajtani, ha nem történik hiba.

# Ebben a példában a try blokk nem generál hibát:
try:
  print("Hello")
except:
  print("Valami elromlott")
else:
  print("Nem történt semmi baj")

Finally

A finally blokk ha meg van adva, végrehajtásra kerül függetlenül attól, hogy a try blokk hibát okoz-e vagy sem.

try:
  print(x)
except:
  print("Valami elromlott")
finally:
  print("A 'try except' befejeződött")

Ez hasznos lehet az objektumok bezárásához és az erőforrások megtisztításához.

# Próbáljon meg megnyitni és írni egy nem írható fájlt:
try:
  f = open("demofile.txt")
  try:
    f.write("Lorum Ipsum")
  except:
    print("Valami hiba történt a fájlba írás közben")
  finally:
  f.close()
except:
  print("Valami hiba történt a fájl megnyitásakor")
# A program a fájlobjektum nyitva hagyása nélkül folytatható.

Kivétel említése

Python-fejlesztőként dönthetsz úgy, hogy kivételt dobsz, ha egy feltétel előfordul.
Kivétel dobásához (vagy emeléséhez) használd a raise kulcsszót.

# Hiba emelése és a program leállítása, ha x kisebb, mint 0:
x = -1
if x < 0:
  raise Exception("Bocsi, szám 0 alatt nem engedélyezett")

Meghatározhatod, hogy milyen hibát jelezzen, és milyen szöveget nyomtatjon ki a felhasználónak.

# Emelje fel a TypeError kivételt, ha az x nem egész szám:
x = "hello"
if not type(x) is int:
  raise TypeError("Csak egész számok megengedettek")
